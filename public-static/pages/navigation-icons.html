Usage Guide & Technical Specs

  • Adaptive Colors: Icons automatically switch between blue and white based on theme
  • Transparent Backgrounds: All icons use glassmorphism with backdrop-filter blur effects
  • Gradient Support: Built-in SVG gradients for enhanced visual depth
  • Scalable: Designed for 16px to 128px+ sizes with perfect clarity
  • Context-Aware: Dark mode uses blue accents, light mode uses deeper blues
  • Modern Effects: Hover animations, shadow effects, and smooth transitions
  • Copy-Ready: Click any icon to copy its complete data including styles
  • Framework Agnostic: Works with React, Vue, Angular, and vanilla HTML/CSS
